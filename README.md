# Smart Assistant for Research Summarization

A GenAI-powered document assistant for summarization, Q&A, and logic-based evaluation.

## 🚀 Features

- 📄 PDF/TXT upload
- 🔍 Auto-summary
- 🧠 Ask Anything
- ❓ Challenge Me mode
- 🧾 Justified answers from document
- ✨ Built with LangChain + Streamlit + OpenAI
